# 📄 Розділ 4: РОБОТА З ТЕКСТОМ

## 📌 🔍 grep — пошук у тексті
| Команда            | Для чого використовується | Приклад                  | Що виведе       |
| ------------------ | ------------------------- | ------------------------ | --------------- |
| `grep "text" file` | пошук рядка               | `grep "error" log.txt`   | рядки з "error" |
| `grep -i`          | без регістру              | `grep -i error log.txt`  | Error/error     |
| `grep -r`          | рекурсивно                | `grep -r error /var/log` | у всіх файлах   |
| `grep -n`          | з номером рядка           | `grep -n error file`     | `5:error`       |
| `grep -v`          | інверсія                  | `grep -v error file`     | без "error"     |
| `grep -c`          | кількість рядків          | `grep -c error file`     | число           |
| `grep -l`          | тільки імена файлів       | `grep -l error *.log`    | список          |
| `grep -w`          | точне слово               | `grep -w error file`     | тільки слово    |
| `grep -E`          | regex (розширений)        | `grep -E "err \| fail"`  | або             |
| `grep -o`          | тільки збіг               | `grep -o "error"`        | тільки слово    |
| `grep -A 2`        | 2 рядки після             | `grep -A 2 error file`   | контекст        |
| `grep -B 2`        | 2 рядки до                | `grep -B 2 error file`   | контекст        |
| `grep -C 2`        | контекст                  | `grep -C 2 error file`   | до/після        |

## 📌 ✂️ cut — вирізання частин тексту
| Команда               | Для чого використовується | Приклад               | Що виведе  |
| --------------------- | ------------------------- | --------------------- | ---------- |
| `cut -d ":" -f1 file` | поле по роздільнику       | `/etc/passwd`         | user       |
| `cut -c 1-5 file`     | символи                   | `cut -c 1-5 file.txt` | перші 5    |
| `cut -d "," -f2,3`    | кілька полів              | CSV                   | поля 2 і 3 |

## 📌 🔀 sort — сортування
| Команда       | Для чого використовується | Приклад           | Що виведе    |
| ------------- | ------------------------- | ----------------- | ------------ |
| `sort file`   | сортування                | `sort file.txt`   | за алфавітом |
| `sort -r`     | у зворотному порядку      | `sort -r`         | reverse      |
| `sort -n`     | числове                   | `sort -n file`    | 1,2,10       |
| `sort -k 2`   | по колонці                | `sort -k 2 file`  | по 2 колонці |
| `sort -t ":"` | роздільник                | `sort -t ":" -k1` | по полю      |
| `sort -u`     | унікальні                 | `sort -u file`    | без повторів |

## 📌 🔁 uniq — унікальні рядки
| Команда   | Для чого використовується | Приклад             | Що виведе    |
| --------- | ------------------------- | ------------------- | ------------ |
| `uniq`    | прибирає дублікати        | `sort file \| uniq` | унікальні    |
| `uniq -c` | підрахунок                | `uniq -c`           | кількість    |
| `uniq -d` | тільки дублікати          | `uniq -d`           | повтори      |
| `uniq -u` | тільки унікальні          | `uniq -u`           | без повторів |

## 📌 🔄 tr — заміна символів
| Команда       | Для чого використовується | Приклад                 | Що зробить        |
| ------------- | ------------------------- | ----------------------- | ----------------- |
| `tr a-z A-Z`  | lower → upper             | `echo hi \| tr a-z A-Z` | HI                |
| `tr -d`       | видалення                 | `echo hi! \| tr -d '!'` | hi                |
| `tr -s`       | стискання                 | `tr -s ' '`             | прибере дублікати |
| `tr ' ' '\n'` | розбиття                  | слова → рядки           | список            |

## 📌 🧠 awk — обробка тексту (поля)
| Команда                           | Для чого використовується | Приклад                 | Що виведе |
| --------------------------------- | ------------------------- | ----------------------- | --------- |
| `awk '{print $1}' file`           | перше поле                | `awk '{print $1}' file` | колонка 1 |
| `awk '{print $1,$3}'`             | кілька полів              | `awk '{print $1,$3}'`   | 1 і 3     |
| `awk -F ":"`                      | роздільник                | `/etc/passwd`           | поля      |
| `awk '{print NF}'`                | кількість полів           | `awk '{print NF}'`      | число     |
| `awk '{sum+=$1} END {print sum}'` | сума                      | файл чисел              | сума      |
| `awk '$1 > 10'`                   | фільтр                    | `awk '$1 > 10'`         | рядки     |

## 📌 🧬 sed — потокове редагування
| Команда          | Для чого використовується | Приклад                    | Що зробить |
| ---------------- | ------------------------- | -------------------------- | ---------- |
| `sed 's/a/b/'`   | заміна                    | `sed 's/error/warn/' file` | 1 заміна   |
| `sed 's/a/b/g'`  | всі входження             | `sed 's/error/warn/g'`     | всі        |
| `sed -i`         | змінити файл              | `sed -i 's/a/b/g' file`    | inplace    |
| `sed -n 'p'`     | друк                      | `sed -n '1,5p' file`       | 1–5 рядки  |
| `sed 'd'`        | видалення                 | `sed '1d' file`            | без 1      |
| `sed '/error/d'` | видалити рядки            | `sed '/error/d'`           | без error  |

## 📌 🔗 Комбінації (pipeline)
| Команда               | Для чого використовується | Приклад                               | Що зробить |
| --------------------- | ------------------------- | ------------------------------------- | ---------- |
| `grep \| sort`        | пошук + сортування        | `grep error log \| sort`              | відсортує  |
| `cut \| sort \| uniq` | статистика                | `cut -d: -f1 file \| sort \| uniq -c` | підрахунок |
| `ps \| grep`          | пошук процесів            | `ps aux \| grep nginx`                | знайде     |
| `cat \| awk`          | обробка                   | `cat file \| awk '{print $1}'`        | колонка    |

## ✅ Підсумок

Це ядро роботи з текстом:

- 🔍 grep — знайти
- ✂️ cut — вирізати
- 🔀 sort — відсортувати
- 🔁 uniq — прибрати дублікати
- 🔄 tr — замінити символи
- 🧠 awk — логіка і обробка
- 🧬 sed — редагування
