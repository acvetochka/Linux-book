Структура довідки Linux команд

Ось логічна структура, яку будемо поступово заповнювати:

1. 🧩 Базові команди (основи роботи в терміналі)
   - навігація по файловій системі
   - перегляд вмісту
   - базові дії з файлами
2. 📁 Файлова система
    - робота з файлами та директоріями
    - права доступу
    - посилання (links)
3. 👤 Користувачі та групи
    - створення, зміна, видалення
    - sudo, права доступу
4. 📄 Робота з текстом
    - перегляд файлів
    - пошук
    - обробка (grep, awk, sed, sort)
5. 📦 Архіви стиснення
    - tar, gzip, zip
6. 📦 Менеджер пакетів (dnf)
    - встановлення, оновлення, видалення
7. 🖥️ Інформація про систему та апаратне забезпечення
    - CPU, RAM, диск
8. ⚙️ Процеси
    - перегляд, керування
9. 🧰 Система (служби, systemd)
    - systemctl, journalctl
10. 🌐 Мережі
    - ip, ping, ss, netstat
11. 🔐 Віддалений доступ
    - ssh, scp, rsync
