# 📦 Розділ 5: АРХІВИ ТА СТИСНЕННЯ

## 📌 📦 tar — архівація (об’єднання файлів)

👉 tar сам по собі не стискає, а лише створює архів  
(але часто використовується разом зі стисненням)

### 🔹 Основні операції
| Команда                    | Для чого використовується | Приклад                  | Що зробить   |
| -------------------------- | ------------------------- | ------------------------ | ------------ |
| `tar -cf archive.tar file` | створити архів            | `tar -cf a.tar file.txt` | створить     |
| `tar -xf archive.tar`      | розпакувати               | `tar -xf a.tar`          | витягне      |
| `tar -tf archive.tar`      | список                    | `tar -tf a.tar`          | покаже вміст |

### 🔹 Часті ключі
| Ключ | Значення        |
| ---- | --------------- |
| `-c` | create          |
| `-x` | extract         |
| `-t` | list            |
| `-f` | файл            |
| `-v` | verbose         |
| `-C` | інша директорія |

### 🔹 Приклади
| Команда                  | Для чого використовується | Приклад                  | Що зробить |
| ------------------------ | ------------------------- | ------------------------ | ---------- |
| `tar -cvf a.tar dir`     | архів папки               | `tar -cvf a.tar folder/` | створить   |
| `tar -xvf a.tar`         | розпакування              | `tar -xvf a.tar`         | витягне    |
| `tar -xvf a.tar -C /tmp` | інша папка                | команда                  | у `/tmp`   |

## 📌 🗜️ Стиснення через tar
### 🔹 gzip (.tar.gz / .tgz)
| Команда                 | Для чого використовується | Приклад | Що зробить        |
| ----------------------- | ------------------------- | ------- | ----------------- |
| `tar -czf a.tar.gz dir` | створити gzip             | команда | архів + стиснення |
| `tar -xzf a.tar.gz`     | розпакувати               | команда | витягне           |
| `tar -tzf a.tar.gz`     | список                    | команда | вміст             |

### 🔹 bzip2 (.tar.bz2)
| Команда                  | Для чого використовується | Приклад | Що зробить         |
| ------------------------ | ------------------------- | ------- | ------------------ |
| `tar -cjf a.tar.bz2 dir` | створити                  | команда | сильніше стиснення |
| `tar -xjf a.tar.bz2`     | розпакувати               | команда | витягне            |

### 🔹 xz (.tar.xz)
| Команда                 | Для чого використовується | Приклад | Що зробить         |
| ----------------------- | ------------------------- | ------- | ------------------ |
| `tar -cJf a.tar.xz dir` | створити                  | команда | найкраще стиснення |
| `tar -xJf a.tar.xz`     | розпакувати               | команда | витягне            |

## 📌 🗜️ Окремі утиліти стиснення
### 🔹 gzip
| Команда           | Для чого використовується | Приклад         | Що зробить    |
| ----------------- | ------------------------- | --------------- | ------------- |
| `gzip file`       | стиснути                  | `gzip file.txt` | → file.txt.gz |
| `gzip -d file.gz` | розпакувати               | команда         | відновить     |
| `gzip -k`         | залишити оригінал         | команда         | не видалить   |
| `gzip -r`         | рекурсивно                | команда         | папка         |

### 🔹 bzip2
| Команда             | Для чого використовується | Приклад          | Що зробить |
| ------------------- | ------------------------- | ---------------- | ---------- |
| `bzip2 file`        | стиснути                  | `bzip2 file.txt` | → .bz2     |
| `bzip2 -d file.bz2` | розпакувати               | команда          | файл       |

### 🔹 xz
| Команда         | Для чого використовується | Приклад       | Що зробить |
| --------------- | ------------------------- | ------------- | ---------- |
| `xz file`       | стиснути                  | `xz file.txt` | → .xz      |
| `xz -d file.xz` | розпакувати               | команда       | файл       |

## 📌 📦 zip / unzip
### 🔹 zip
| Команда                  | Для чого використовується | Приклад               | Що зробить |
| ------------------------ | ------------------------- | --------------------- | ---------- |
| `zip archive.zip file`   | створити zip              | `zip a.zip file.txt`  | архів      |
| `zip -r archive.zip dir` | папка                     | `zip -r a.zip folder` | рекурсивно |

### 🔹 unzip
| Команда                    | Для чого використовується | Приклад       | Що зробить |
| -------------------------- | ------------------------- | ------------- | ---------- |
| `unzip archive.zip`        | розпакувати               | `unzip a.zip` | витягне    |
| `unzip -l archive.zip`     | список                    | команда       | покаже     |
| `unzip -d dir archive.zip` | інша папка                | команда       | в dir      |

## 📌 🔥 Порівняння форматів
| Формат | Розмір    | Швидкість | Використання  |
| ------ | --------- | --------- | ------------- |
| `.gz`  | середній  | швидкий   | стандарт      |
| `.bz2` | менший    | повільний | старі системи |
| `.xz`  | найменший | повільний | архіви        |
| `.zip` | середній  | швидкий   | Windows       |

## 📌 🔗 Типові сценарії
| Задача             | Команда                       |
| ------------------ | ----------------------------- |
| Backup папки       | `tar -czf backup.tar.gz dir/` |
| Розпакувати backup | `tar -xzf backup.tar.gz`      |
| Стиснути лог       | `gzip log.txt`                |
| Розпакувати zip    | `unzip file.zip`              |

## ✅ Підсумок
- `tar` → архівує
- `gzip` / `bzip2` / `xz` → стискають
- `zip` → універсальний формат
