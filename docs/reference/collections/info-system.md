# 🖥️ Розділ 7: ІНФОРМАЦІЯ ПРО СИСТЕМУ ТА АПАРАТНЕ ЗАБЕЗПЕЧЕННЯ

## 📌 🧠 Загальна інформація про систему
| Команда       | Для чого використовується | Приклад       | Що виведе     |
| ------------- | ------------------------- | ------------- | ------------- |
| `uname`       | інформація про ядро       | `uname`       | Linux         |
| `uname -a`    | повна інформація          | `uname -a`    | kernel, arch  |
| `hostname`    | ім’я хоста                | `hostname`    | server1       |
| `hostnamectl` | детальна інформація       | `hostnamectl` | OS, kernel    |
| `uptime`      | час роботи                | `uptime`      | uptime + load |

## 📌 🧠 CPU
| Команда             | Для чого використовується | Приклад | Що виведе      |
| ------------------- | ------------------------- | ------- | -------------- |
| `lscpu`             | інформація про CPU        | `lscpu` | cores, threads |
| `cat /proc/cpuinfo` | детально                  | команда | повний опис    |
| `nproc`             | кількість ядер            | `nproc` | число          |

## 📌 🧠 RAM (пам’ять)
| Команда   | Для чого використовується | Приклад  | Що виведе |
| --------- | ------------------------- | -------- | --------- |
| `free`    | використання RAM          | `free`   | bytes     |
| `free -h` | у зручному форматі        | команда  | MB/GB     |
| `vmstat`  | статистика                | `vmstat` | RAM + CPU |

## 📌 💽 Диски та файлові системи
| Команда    | Для чого використовується | Приклад        | Що виведе       |
| ---------- | ------------------------- | -------------- | --------------- |
| `lsblk`    | список дисків             | `lsblk`        | диски/розділи   |
| `lsblk -f` | з FS                      | команда        | файлові системи |
| `df -h`    | використання              | `df -h`        | заповнення      |
| `du -sh`   | розмір папки              | `du -sh /home` | розмір          |
| `blkid`    | UUID                      | команда        | UUID            |
| `mount`    | змонтовані FS             | команда        | список          |

## 📌 🧾 Інформація про ОС
| Команда               | Для чого використовується | Приклад | Що виведе |
| --------------------- | ------------------------- | ------- | --------- |
| `cat /etc/os-release` | інформація про ОС         | команда | distro    |
| `lsb_release -a`      | distro info               | команда | версія    |

## 📌 🔌 Апаратне забезпечення
| Команда  | Для чого використовується | Приклад | Що виведе    |
| -------- | ------------------------- | ------- | ------------ |
| `lspci`  | PCI пристрої              | `lspci` | GPU, network |
| `lsusb`  | USB пристрої              | `lsusb` | USB          |
| `lsmem`  | пам’ять                   | команда | RAM блоки    |
| `hwinfo` | повна інфо                | команда | деталі       |

## 📌 🌡️ Навантаження системи
| Команда  | Для чого використовується | Приклад | Що виведе |
| -------- | ------------------------- | ------- | --------- |
| `top`    | процеси в реальному часі  | `top`   | live      |
| `htop`   | покращений top            | `htop`  | UI        |
| `uptime` | load average              | команда | load      |
| `w`      | активні користувачі       | команда | users     |

## 📌 📊 Додаткові утиліти
| Команда                | Для чого використовується | Приклад            | Що виведе    |
| ---------------------- | ------------------------- | ------------------ | ------------ |
| `watch`                | повтор команди            | `watch -n 1 df -h` | оновлення    |
| `dmesg`                | kernel log                | `dmesg`            | повідомлення |
| `dmesg  \| grep error` | помилки                   | команда            | error        |

## 📌 🔥 Типові сценарії
| Задача             | Команда    |
| ------------------ | ---------- |
| Перевірити CPU     | `lscpu`    |
| Перевірити RAM     | `free -h`  |
| Перевірити диски   | `lsblk`    |
| Перевірити місце   | `df -h`    |
| Подивитись систему | `uname -a` |
| Логи ядра          | `dmesg`    |

📌 🧠 Важливо знати

👉 /proc — псевдофайлова система:
- /proc/cpuinfo
- /proc/meminfo  
👉 /sys — інформація про пристрої

## ✅ Підсумок
- CPU → lscpu
- RAM → free
- диск → lsblk, df
- система → uname, hostnamectl
- hardware → lspci, lsusb
- навантаження → top, uptime
