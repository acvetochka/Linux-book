# Розділ 12: ВАЖЛИВІ КОНФІГУРАЦІЙНІ ФАЙЛИ ТА ДЕ ВОНИ ЗНАХОДЯТЬСЯ

## 📌 🧭 1. Загальна структура Linux (FHS)
| Шлях            | Що містить                          |
| --------------- | ----------------------------------- |
| `/etc`          | конфігурація системи                |
| `/var`          | змінні дані (логи, кеш)             |
| `/home`         | домашні директорії                  |
| `/root`         | home root                           |
| `/usr`          | програми та бібліотеки              |
| `/bin`, `/sbin` | основні утиліти                     |
| `/proc`         | інформація про систему (віртуальна) |
| `/sys`          | пристрої                            |
| `/tmp`          | тимчасові файли                     |

## 📌 👤 2. Користувачі та аутентифікація
| Файл              | Що містить                     |
| ----------------- | ------------------------------ |
| `/etc/passwd`     | користувачі (UID, shell, home) |
| `/etc/shadow`     | паролі (хеші)                  |
| `/etc/group`      | групи                          |
| `/etc/gshadow`    | паролі груп                    |
| `/etc/login.defs` | політика паролів               |
| `/etc/skel/`      | шаблон для нових користувачів  |

## 📌 🔐 3. sudo і права
| Файл              | Що містить        |
| ----------------- | ----------------- |
| `/etc/sudoers`    | правила sudo      |
| `/etc/sudoers.d/` | додаткові конфіги |

📌 редагувати тільки через:
```bash
visudo
```

## 📌 🌐 4. Мережа
### 🔹 NetworkManager
| Файл                                      | Що містить           |
| ----------------------------------------- | -------------------- |
| `/etc/NetworkManager/NetworkManager.conf` | основні налаштування |
| `/etc/NetworkManager/system-connections/` | профілі мереж        |

### 🔹 DNS / hostname
| Файл               | Що містить   |
| ------------------ | ------------ |
| `/etc/hosts`       | локальні DNS |
| `/etc/resolv.conf` | DNS сервери  |
| `/etc/hostname`    | ім’я хоста   |

## 📌 ⚙️ 5. Сервіси (systemd)
| Шлях                       | Що містить       |
| -------------------------- | ---------------- |
| `/etc/systemd/system/`     | кастомні юніти   |
| `/usr/lib/systemd/system/` | стандартні юніти |
| `/run/systemd/system/`     | runtime          |

## 📌 📜 6. Логи
| Шлях                | Що містить |
| ------------------- | ---------- |
| `/var/log/`         | всі логи   |
| `/var/log/messages` | загальні   |
| `/var/log/secure`   | auth       |
| `/var/log/dnf.log`  | пакети     |

📌 journalctl:
- логи також у binary форматі

## 📌 📦 7. Пакети (dnf / rpm)
| Шлях                | Що містить  |
| ------------------- | ----------- |
| `/etc/yum.repos.d/` | репозиторії |
| `/var/cache/dnf/`   | кеш         |
| `/var/lib/rpm/`     | база rpm    |

## 📌 🖥️ 8. Завантаження системи (boot)
| Файл                   | Що містить        |
| ---------------------- | ----------------- |
| `/boot/`               | kernel, initramfs |
| `/boot/grub2/grub.cfg` | grub конфіг       |
| `/etc/default/grub`    | налаштування grub |

## 📌 ⏰ 9. Планувальники
| Файл               | Що містить     |
| ------------------ | -------------- |
| `/etc/crontab`     | системний cron |
| `/etc/cron.d/`     | cron jobs      |
| `/var/spool/cron/` | user cron      |

## 📌 🔐 10. SSH
| Файл                     | Що містить           |
| ------------------------ | -------------------- |
| `/etc/ssh/sshd_config`   | налаштування сервера |
| `~/.ssh/config`          | клієнт               |
| `~/.ssh/authorized_keys` | ключі доступу        |

## 📌 🧠 11. Shell і середовище
| Файл              | Що містить             |
| ----------------- | ---------------------- |
| `/etc/profile`    | глобальні налаштування |
| `/etc/bashrc`     | bash                   |
| `~/.bashrc`       | користувач             |
| `~/.bash_profile` | login shell            |

## 📌 🔥 12. Важливі runtime місця
| Шлях    | Що містить      |
| ------- | --------------- |
| `/run/` | тимчасові дані  |
| `/tmp/` | тимчасові файли |
| `/dev/` | пристрої        |

## 📌 🧠 Як це все пов’язано (логіка системи)
🔹 При створенні користувача:
- /etc/passwd
- /etc/shadow
- /home/user
- /etc/skel

🔹 При встановленні пакета:
- /usr/bin
- /etc
- /var/lib/rpm

🔹 При запуску сервісу:
- /usr/lib/systemd/system
- /etc/systemd/system
- /var/log

🔹 При мережі:
- /etc/NetworkManager
- /etc/resolv.conf
- /etc/hosts

## 📌 🧠 Важливі принципи

👉 /etc → конфіг 
👉 /var → змінні дані  
👉 /usr → програми  
👉 /home → користувач  

## ✅ Підсумок

Це карта системи:

- 👤 користувачі → /etc/passwd
- 🔐 права → /etc/sudoers
- 🌐 мережа → /etc/NetworkManager
- 📦 пакети → /var/lib/rpm
- ⚙️ сервіси → /etc/systemd
- 📜 логи → /var/log
- 🔑 ssh → /etc/ssh
