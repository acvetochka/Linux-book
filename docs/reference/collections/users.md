# 👤 Розділ 3: КОРИСТУВАЧІ ТА ГРУПИ

## 📌 👤 Інформація про користувача
| Команда   | Для чого використовується | Приклад  | Що виведе     |
| --------- | ------------------------- | -------- | ------------- |
| `whoami`  | поточний користувач       | `whoami` | username      |
| `id`      | UID, GID, групи           | `id`     | uid=1000(...) |
| `id user` | інформація про іншого     | `id bob` | дані          |
| `who`     | хто увійшов               | `who`    | користувачі   |
| `w`       | активність користувачів   | `w`      | хто що робить |

## 📌 ➕ Створення користувачів
| Команда                | Для чого використовується | Приклад          | Що зробить  |
| ---------------------- | ------------------------- | ---------------- | ----------- |
| `useradd user`         | створити користувача      | `useradd bob`    | створить    |
| `useradd -m user`      | з home директорією        | `useradd -m bob` | `/home/bob` |
| `useradd -s /bin/bash` | shell                     | команда          | bash        |
| `useradd -u 1001`      | UID                       | команда          | UID=1001    |
| `useradd -g group`     | primary group             | команда          | група       |
| `useradd -G g1,g2`     | додаткові групи           | команда          | групи       |


📌 Після створення:
```bash
passwd bob
```

## 📌 🔑 Паролі
| Команда          | Для чого використовується | Приклад      | Що зробить |
| ---------------- | ------------------------- | ------------ | ---------- |
| `passwd`         | змінити пароль            | `passwd`     | для себе   |
| `passwd user`    | іншому                    | `passwd bob` | змінить    |
| `passwd -l user` | заблокувати               | команда      | lock       |
| `passwd -u user` | розблокувати              | команда      | unlock     |

## 📌 ✏️ Зміна користувача
| Команда                  | Для чого використовується | Приклад | Що зробить |
| ------------------------ | ------------------------- | ------- | ---------- |
| `usermod -l new old`     | змінити ім’я              | команда | rename     |
| `usermod -d /home/new`   | home                      | команда | новий шлях |
| `usermod -aG group user` | додати в групу            | команда | append     |
| `usermod -s /bin/zsh`    | shell                     | команда | змінить    |


📌 Важливо:
- -aG → НЕ перезаписує групи

## 📌 ❌ Видалення користувачів
| Команда           | Для чого використовується | Приклад       | Що зробить      |
| ----------------- | ------------------------- | ------------- | --------------- |
| `userdel user`    | видалити                  | `userdel bob` | видалить        |
| `userdel -r user` | з home                    | команда       | разом з файлами |

## 📌 👥 Групи
### 🔹 Створення і керування
| Команда               | Для чого використовується | Приклад        | Що зробить |
| --------------------- | ------------------------- | -------------- | ---------- |
| `groupadd group`      | створити                  | `groupadd dev` | створить   |
| `groupdel group`      | видалити                  | команда        | видалить   |
| `groupmod -n new old` | перейменувати             | команда        | rename     |

### 🔹 Робота з групами
| Команда        | Для чого використовується | Приклад  | Що зробить |
| -------------- | ------------------------- | -------- | ---------- |
| `groups`       | групи користувача         | `groups` | список     |
| `groups user`  | іншого                    | команда  | список     |
| `newgrp group` | змінити групу             | команда  | нова group |

## 📌 🔐 sudo (права адміністратора)
| Команда        | Для чого використовується | Приклад            | Що зробить |
| -------------- | ------------------------- | ------------------ | ---------- |
| `sudo command` | виконати як root          | `sudo dnf install` | root       |
| `sudo -i`      | shell root                | команда            | root shell |
| `sudo -l`      | дозволи                   | команда            | список     |

### 🔹 Додавання до sudo
| Команда                  | Для чого використовується | Приклад | Що зробить |
| ------------------------ | ------------------------- | ------- | ---------- |
| `usermod -aG wheel user` | доступ до sudo            | команда | sudo права |

📌 У Fedora/RHEL:
- група wheel = sudo

## 📌 📂 Системні файли
| Файл          | Що містить  |
| ------------- | ----------- |
| `/etc/passwd` | користувачі |
| `/etc/shadow` | паролі      |
| `/etc/group`  | групи       |

## 📌 🔥 Типові сценарії
| Задача               | Команда                 |
| -------------------- | ----------------------- |
| Створити користувача | `useradd -m bob`        |
| Задати пароль        | `passwd bob`            |
| Додати в sudo        | `usermod -aG wheel bob` |
| Подивитись UID       | `id bob`                |
| Видалити користувача | `userdel -r bob`        |

## 📌 🧠 Важливі нюанси

👉 UID:
- 0 → root
- 1000+ → звичайні користувачі

👉 primary vs secondary group:
- primary → -g
- secondary → -G

## ✅ Підсумок
- створення → useradd
- зміна → usermod
- видалення → userdel
- групи → groupadd
- права → sudo, wheel
