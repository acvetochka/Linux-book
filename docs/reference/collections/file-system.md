# 📁 Розділ 2: ФАЙЛОВА СИСТЕМА

## 📌 1. Права доступу (permissions)
### 🔐 chmod — зміна прав доступу
| Команда            | Для чого використовується     | Приклад               | Що зробить         |
| ------------------ | ----------------------------- | --------------------- | ------------------ |
| `chmod 755 file`   | задати права (числовий режим) | `chmod 755 script.sh` | rwxr-xr-x          |
| `chmod +x file`    | додати виконання              | `chmod +x script.sh`  | зробить executable |
| `chmod -w file`    | прибрати запис                | `chmod -w file.txt`   | заборонить запис   |
| `chmod u+x file`   | додати для власника           | `chmod u+x file`      | тільки user        |
| `chmod g-w file`   | забрати у групи               | `chmod g-w file`      | group без write    |
| `chmod o+r file`   | доступ іншим                  | `chmod o+r file`      | others read        |
| `chmod -R 755 dir` | рекурсивно                    | `chmod -R 755 dir`    | для всієї папки    |

📌 Позначення:
- u — user
- g — group
- o — others
- rwx — read/write/execute

### 👤 chown — зміна власника
| Команда                 | Для чого використовується | Приклад                  | Що зробить      |
| ----------------------- | ------------------------- | ------------------------ | --------------- |
| `chown user file`       | змінити власника          | `chown bob file.txt`     | власник bob     |
| `chown user:group file` | власник + група           | `chown bob:dev file.txt` | user+group      |
| `chown :group file`     | тільки група              | `chown :dev file.txt`    | група dev       |
| `chown -R user dir`     | рекурсивно                | `chown -R bob dir`       | для всієї папки |

### 👥 chgrp — зміна групи
| Команда            | Для чого використовується | Приклад              | Що зробить |
| ------------------ | ------------------------- | -------------------- | ---------- |
| `chgrp dev file`   | змінює групу              | `chgrp dev file.txt` | група dev  |
| `chgrp -R dev dir` | рекурсивно                | `chgrp -R dev dir`   | для папки  |

## 📌 2. Спеціальні права
| Тип        | Опис                         | Приклад          |
| ---------- | ---------------------------- | ---------------- |
| SUID       | виконання від імені власника | `chmod u+s file` |
| SGID       | виконання від імені групи    | `chmod g+s dir`  |
| Sticky bit | тільки власник може видаляти | `chmod +t /tmp`  |

## 📌 3. Посилання (links)
### 🔗 ln — створення посилань
| Команда            | Для чого використовується | Приклад                   | Що зробить      |
| ------------------ | ------------------------- | ------------------------- | --------------- |
| `ln file link`     | hard link                 | `ln file.txt link.txt`    | той самий inode |
| `ln -s file link`  | symbolic link             | `ln -s file.txt link.txt` | "ярлик"         |
| `ln -sf file link` | перезапис symlink         | `ln -sf new.txt link.txt` | оновить         |

📌 Різниця:

- hard link → той самий inode
- symlink → шлях до файлу

## 📌 4. Пошук файлів
### 🔍 find — потужний пошук
| Команда                  | Для чого використовується | Приклад                  | Що знайде    |
| ------------------------ | ------------------------- | ------------------------ | ------------ |
| `find . -name file.txt`  | пошук по імені            | `find / -name file.txt`  | файл         |
| `find . -iname file.txt` | без регістру              | `find . -iname FILE.txt` | файл         |
| `find . -type f`         | тільки файли              | `find . -type f`         | всі файли    |
| `find . -type d`         | тільки папки              | `find . -type d`         | директорії   |
| `find . -size +10M`      | більші за 10MB            | `find . -size +10M`      | великі файли |
| `find . -user bob`       | по користувачу            | `find . -user bob`       | файли bob    |
| `find . -mtime -1`       | змінені за день           | `find . -mtime -1`       | за 24h       |

### ⚡ find з діями
| Команда   | Для чого використовується | Приклад                               | Що зробить |
| --------- | ------------------------- | ------------------------------------- | ---------- |
| `-exec`   | виконати команду          | `find . -name "*.log" -exec rm {} \;` | видалить   |
| `-delete` | видалити                  | `find . -name "*.tmp" -delete`        | видалить   |
| `-print`  | вивід                     | `find . -name "*.txt"`                | список     |

### 📌 🔗 xargs — передача stdin як аргументів
| Команда         | Для чого використовується    | Приклад                                   | Що зробить           |
| --------------- | ---------------------------- | ----------------------------------------- | -------------------- |
| `xargs`         | перетворює stdin в аргументи | `echo "a b c" \| xargs echo`              | `a b c`              |
| `xargs rm`      | масове видалення             | `cat files.txt \| xargs rm`               | видалить файли       |
| `xargs -n 1`    | по одному аргументу          | `echo "a b c" \| xargs -n 1 echo`         | по одному в рядок    |
| `xargs -n 5`    | групування                   | `seq 1 10 \| xargs -n 5 echo`             | по 5 елементів       |
| `xargs -P 4`    | паралельне виконання         | `... \| xargs -P 4 gzip`                  | 4 процеси            |
| `xargs -I {}`   | placeholder                  | `seq 1 3 \| xargs -I {} touch file{}.txt` | file1, file2         |
| `xargs -d '\n'` | роздільник newline           | `cat list.txt \| xargs -d '\n' rm`        | безпечніше           |
| `xargs -0`      | робота з NULL                | `find . -print0 \| xargs -0 rm`           | безпечно з пробілами |

📌 Типові комбінації
| Команда             | Для чого використовується    | Приклад                              | Що зробить |
| ------------------- | ---------------------------- | ------------------------------------ | ---------- |
| `find ... \| xargs` | масова обробка               | `find . -name "*.log" \| xargs rm`   | видалить   |
| `grep ... \| xargs` | робота з результатами пошуку | `grep -rl "error" . \| xargs rm`     | видалить   |
| `seq ... \| xargs`  | генерація                    | `seq 1 3 \| xargs -I {} echo file{}` | file1…     |

### 🔎 locate — швидкий пошук
Команда	Для чого використовується	Приклад	Що виведе
locate file.txt	швидкий пошук	locate file.txt	список шляхів
locate -i file	без регістру	locate -i file	знайде
updatedb	оновити базу	sudo updatedb	оновить

📌 Важливо:
- працює через базу (може бути неактуально)

## 📌 5. Інформація про файли та inode
### inode (концепція)
| Команда     | Для чого використовується | Приклад          | Що виведе    |
| ----------- | ------------------------- | ---------------- | ------------ |
| `ls -i`     | показ inode               | `ls -i file.txt` | номер inode  |
| `stat file` | деталі inode              | `stat file.txt`  | всі метадані |

📌 inode містить:
- права
- власника
- розмір
- часи
- НЕ містить ім'я файлу

## 📌 6. Монтування файлових систем
💽 mount / umount
| Команда                | Для чого використовується | Приклад                        | Що зробить      |
| ---------------------- | ------------------------- | ------------------------------ | --------------- |
| `mount`                | показ змонтованих         | `mount`                        | список          |

## 📌 7. Атрибути файлів (advanced)

| Команда          | Для чого використовується | Приклад              | Що зробить       |
| ---------------- | ------------------------- | -------------------- | ---------------- |
| `chattr +i file` | immutable                 | `chattr +i file.txt` | не можна змінити |
