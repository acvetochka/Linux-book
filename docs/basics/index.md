# Основи Linux

Цей розділ охоплює фундаментальні теми, необхідні для розуміння Linux та подальшого вивчення системного адміністрування.

Тут зібрано як теоретичну основу (архітектура, компоненти системи), так і практичні аспекти роботи: взаємодія через термінал, використання shell та автоматизація за допомогою скриптів.

## Структура розділу

### 01. Fundamentals

Базові концепції Linux та його архітектури.

У цьому підрозділі розглядаються:

- структура операційної системи Linux;
- відмінності між Linux kernel і GNU tools;
- стандарт POSIX;
- сімейства дистрибутивів Linux;
- основні компоненти операційної системи.

Цей блок формує теоретичну основу для всіх наступних тем.

---

### 02. Environment

Середовища доступу до Linux та способи взаємодії з системою.

Тут описується:

- робота Linux у WSL (Windows Subsystem for Linux);
- термінали та термінальні емулятори;
- TTY та псевдотермінали;
- віддалене підключення до системи;
- робота через SSH.

Цей підрозділ пояснює, як користувач отримує доступ до системи та працює з нею.

---

### 03. Shell

Командна оболонка як основний інтерфейс взаємодії з Linux.

Розглядаються:

- синтаксис команд;
- стандартні потоки (stdin, stdout, stderr);
- оператори та спеціальні символи;
- історія команд;
- змінні середовища;
- налаштування Bash.

Shell є ключовим інструментом для роботи з системою.

---

### 04. Shell Scripting

Автоматизація задач за допомогою Bash-скриптів.

У цьому підрозділі:

- структура Bash-скрипта;
- змінні;
- умовні конструкції;
- цикли;
- функції;
- виконання скриптів.

Скрипти дозволяють автоматизувати адміністративні та системні процеси.

## Для чого цей розділ

Після опрацювання цього блоку ти зможеш:

- розуміти, як влаштована Linux-система;
- працювати з терміналом та shell;
- орієнтуватися в середовищах доступу (WSL, SSH тощо);
- писати базові Bash-скрипти;
- підготуватись до вивчення процесів, файлових систем, мереж та безпеки.
